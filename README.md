# kaguya-clock

『超かぐや姫！』の劇中で登場するデジタル時計の再現

デプロイURL: [https://kaguya-clock.iroirolink.com/](https://kaguya-clock.iroirolink.com/)

## 本編登場シーン

1時間39分30秒頃に、約3秒登場します

## 使いかた

スマホやタブレット端末を横画面にして表示してご使用ください

PWAとしてインストールすると、ステータスバーを非表示にすることが可能です

iOSの場合は、ブラウザの共有メニューからホーム画面への追加を選択し、Webアプリとして追加を行ってください

## 蛇足

本編で登場する時計は、「月」の表示幅が1桁分のみ確保されています

すなわち、10月以降の日付を表示するときには工夫が必要です

0. 実は月の表示は16進数
1. 14セグメントLEDの類であるが、2030年の技術で切り目がほとんど見えなくなっている
2. 2桁表示時のみ何らかの技術でLEDが移動する

の3つの案を検討しました

ここで本編の時計をじっくり観察してみると、「月」の表示に使われているセグメントの形と「日」の表示に使われているセグメントの形が微妙に異なっていることが分かります

つまり「月」の表示桁は7セグメントLEDではない可能性があり得るわけです

したがって本プロジェクトでは、1桁のセグメントLEDに対して2桁の数字を押し込む、2番の解決策を採用しました

あの時計が本来どんな風に動作するのか、気になるところです
